## Formatações

 < b >: Texto em negrito
 < strong >: Texto importante
 < i >: Texto em itálico
 < em >: ênfase no texto
 < small >: texto menor
 < mark >: texto destacado
 < del >: texto excluído
 < ins >: texto inserido
 < sub >: texto subscrito
 < sup >: texto sobrescrito
 < blockquote >: texto citado
 < q >: citação curta
 < abbr >: abreviação
 < address >: define o endereço do autor
 < cite >: define o título do trabalho
